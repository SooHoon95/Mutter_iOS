### 설명 (Description, ≤4000자)
```
마음을 글로만 전하기엔 아쉬운 순간이 있어요.
뮤터는 편지에 음악을 얹어, 열어보는 순간 한 편의 장면처럼 마음이 재생되는 감성 음악 편지 앱입니다.

■ 이렇게 써요
· 테마를 고르고 편지를 씁니다 — 웜 아이보리와 골드, 명조체가 담긴 정갈한 편지지
· 편지에 어울리는 음악 한 곡을 담습니다 — 편지를 열면 자동으로 흐르는 배경음악
· 링크로 보내거나, 연결된 사람에게 바로 전합니다

■ 이런 순간에
· 기념일, 생일, 고백, 축하, 위로 — 말로 다 못한 마음
· 오래 간직하고 싶은 한 통의 편지

■ 안심하고 전해요
· 암호 보호 — 링크에 암호를 걸어 받는 사람만 열 수 있어요
· 예약 공개 — 정해둔 시각에만 편지가 열려요
· 링크 무효화 — 언제든 전달을 취소할 수 있어요

■ 받는 사람은 설치가 필요 없어요
링크만 있으면 웹에서 바로 편지를 열어볼 수 있어, 소중한 사람에게 부담 없이 전할 수 있습니다.

지금, 흘려보내기 아까운 마음을 뮤터로 연출해 보세요.
```

## 10. 심사 리스크 & 완화 (Guideline 대응)

**🟢 1.2 사용자 생성 콘텐츠** — 편지=UGC지만 공개 피드 없이 연결된 1:1 전달. 표준 요구 대비 현황:
- 신고(report): `/legal/takedown` 공개 신고 채널 **있음** ✓
- 게시된 연락처: 문의하기(dkehskeh@gmail.com) **있음** ✓
- 계정·데이터 삭제: 프로필 > 계정 삭제 **있음** ✓
- 사용자 **차단(block)**: 미구현 → 리젝 시 "1:1 상호 초대 기반, 공개 피드 없음"으로 소명하거나 차단 최소 구현 추가.

**🟢 5.2 지식재산권(음원)** — 음악은 **SoundCloud 공식 embed** 재생이며 앱이 오디오를 저장·배포하지 않음. 침해 신고 채널(`/legal/takedown`)로 정당 신고 시 해당 편지 오디오 비활성화. 리스크 낮음.

**🟢 4.8 Sign in with Apple** — Google/Kakao 제공 시 Apple 로그인 필수 → **이미 충족**.

**🟢 5.1.1(v) 계정 삭제** — 앱 내 계정 삭제 필요 → **이미 충족**(프로필 > 계정 삭제).

**🟡 도메인 안정성** — `*.vercel.app` 자동 URL에 편지 링크·AASA·딥링크가 전부 묶여 있음. 커스텀 도메인으로 옮기면 이후 링크 깨짐/브랜딩 문제 예방.
